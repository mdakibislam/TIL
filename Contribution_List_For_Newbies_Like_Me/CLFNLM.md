# Contributing to Free Assets 🎨



Please follow these simple guidelines to add your assets to the collection.
Also feel free to use assets just dont forget to mention us.

---

## 🚀 [TIL](https://github.com/mdakibislam/TIL)


1. **Fork the [Repository](https://github.com/mdakibislam/TIL)** to your own GitHub account.
2. **Create a New Branch** for your asset submission:
   ```bash
   git checkout -b feature/add-my-custom-assets
### or   
```

git checkout -b feature/your-contribution-or-features-or-list
```
